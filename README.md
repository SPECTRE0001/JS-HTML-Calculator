# JS-HTML-Calculator

<h2>Description</h2>
A calculator webstite program that can perform addition, subtraction, multiplication, and division and can take input from the buttons or the keyboard.
<br />


<h2>Languages and Utilities Used</h2>

- <b>JavaScript</b>
- <b>HTML</b>
- <b>CSS</b>
